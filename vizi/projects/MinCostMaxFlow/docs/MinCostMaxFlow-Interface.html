Режим визуализации:

Отображаемые элементы:

  1. Текущий поток
  2. Текущая стоимость потока
  3. Легенда для вершины
  4. Стоимость ребра
  5. Вершина:
    • В верхней половине указано название вершины - число (ее номер),буква S (источник) или буква T (сток)
    • Слева снизу указан потенциал вершины (используемый для алгоритма Дейкстры)
    • Справа снизу указно растояние от источника до этой вершины
  6. Ребро; заливка пропорциональна потоку черз это ребро
  7. Поток через ребро / пропускная способность ребра
  8. Панель с комментарием текущего действия
  9. Элементы управления:
    • При помощи кнопок "<<" и ">>" пользователь может перейти к предыдущему или следущему подробному шагу алгоритма
    • При помощи кнопок "<<<" и ">>>" можно двигаться крупными шагами по основным точкам алгоритма.
    • Кнопка "Рестарт" возвращает алгоритм в исходное состояние
    • Кнопка "Авто" переводит визуализатор в режим, при котором шаги делаются автоматически, время между шагами задается параметром "Задержка"
    • Кнопка "?" отображает краткую информацию о визуализаторе
    • Кнопка "Редактировать" переводит визуализатор в режим редактирования
    • Кнопка "Сохранить/Загрузить" вызывает диалог, с помощью которого можно сохранить состояние алгоритма в файл или загрузить его из файла
    • С помощью выпадающего списка "Примеры" можно выбрать пример, на котором будет работать алгоритм.

Режим редактирования:

Действия, которые можно выполнять в режиме редактирования:

  1. Изменение положения вершины — для этого необходимо "перетащить" вершину
  2. Добавление вершины — "кликните" правой кнопокой мыши на свободное пространсво и в появившемя меню нажмите "Добавить"
  3. Удаление вершины — "кликните" правой кнопкой мыши на вершину, которую требуется удалить, и в появившемся меню нажмите "Удалить" (нельзя удалять источник и сток)
  4. Создание ребра — "кликните" на вершину, из которой будет идти ребро, а затем на вершину, конечную, для этого ребра, в качестве стоимости и пропускной способности будут установлены случайные числа, при этом если такое ребро между этими верщинами уже существовало, оно заменится на новое
  5. Удаление ребра — "кликните" по любому из элементов требуемого ребра (стоимости или пропускной способности) и выберите среди появившихся кнопок "x"
  6. Изменение параметров ребра — "кликните" по требуемому элементу (стоимости или пропускной способности) и среди появившихся кнопок выберете "+" для увеличения и "-" — для уменьшения параметра; чтобы кнопки убрались — нажмите на любое свободное место
  7. Возврат к визуализации — нажмиете "Применить" для сохранения изменения и начала выполнения алгоритма
