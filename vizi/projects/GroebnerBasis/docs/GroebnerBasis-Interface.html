Главное окно

  1. Столбец в левой части экрана с заголовком G1 - идеал G1, использующийся для решения задачи.
  2. Столбец в средней части экрана с заголовком G2 - идеал G2, использующийся для решения задачи.
  3. Полином, выделенный красным цветом - полином, с которым на текущем шаге производится действие.
  4. Кнопки "<<" и ">>" выполняют переход к соответственно предыдущему и следущему подробному шагу алгоритма.
  5. Кнопки "<<<" и ">>>" выполняют переход к соответственно предыдущему и следующему крупному шагу алгоритма. При крупных шагах отоброжаются только основные части алгритма.
  6. Кнопка "Рестарт" возвращает алгоритм в исходное состояние.
  7. Кнопка "Авто" переводит визуализатор в режим автоматической смены шагов. Период устанавливается параметром "Задержка".
  8. Кнопка "?" отображает краткую информацию об авторе визуализатора.
  9. Кнопка "Сохранить/Загрузить" вызывает диалог, с помощью которого задать новые начальные условия, а также сохранить состояние алгоритма в файл или загрузить его из файла.

Диалог сохранения/загрузки

  1. Необходимо указать количество используемых переменных, а также перечислить их. В именах переменных можно использовать любые символы, кроме цифр (в нормальном и верхнем регистрах), символов "^", ".", "+", "-", "*", а также символов-разделителей (пробелов, переводов строк и т.д.)
  2. При набирании полинома также запрещается использовать разделители внутри полинома. Для умножения нужно использовать только "*" (x*y вместо xy, 2*x вместо 2x), для возведения в степень нужно использовать знак "^" (x в степени 5 - x^5). Разрешается опускать "1" в коэффициентах ("x" вместо "1*x", "-y" вместо "-1y")
